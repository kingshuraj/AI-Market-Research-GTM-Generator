# AI Market Research & GTM Strategy Generator

An AI-powered consulting tool that generates structured market intelligence and go-to-market (GTM) strategies.

## Overview
This application allows users to input an industry, company, or market and generates a consulting-style market research report including:

- Market Overview
- Market Size & Growth
- Key Competitors
- Customer Segments
- Go-To-Market Strategy
- Risks & Opportunities

## Tech Stack
- Python
- Streamlit
- OpenAI API

## How It Works
1. User enters an industry or company.
2. The application sends the query to a Large Language Model.
3. The model generates structured consulting-style market insights.

## Example Use Cases
- Market research
- Consulting strategy frameworks
- Startup market validation
- Competitive analysis

## Note
This demo requires OpenAI API credits to generate live insights. Once API credits are added, the tool will automatically generate full market intelligence reports.
